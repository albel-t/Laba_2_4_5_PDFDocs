## PDF документы

- Лабораторная 4.1 (2 балла, + 2 балла): Имеется лог-файл, полученный с "умного" зарядного устройства. Данные в файле хранятся в формате `TSV` значения разделены друг от друга пробелами или табуляцией (может быть один или более разделительных символов подряд) и разделены на секции, каждая секция начинается со строки заголовка. Необходимо написать программу, которая анализирует содержимое файла и генерирует отчёт в формате `PDF`. Что анализировать и какой должен быть результат работы программы, указано в техническом задании приложенном к задаче. Основное распределение баллов указано внутри технического задания.
- Лабораторная 4.2 (2 балла): Разработать программу, которая формирует выходные данные на основе документа  на вход которой поступает готовый документ «Свидетельство о государственной регистрации программы для ЭВМ» в формате PDF. Она извлекает из документа: название, номер регистрации (свидетельства), дата регистрации (публикации), авторов, правообладателя.
- Лабораторная 4.3 (2 балла): Разработать программу, которая на сайте осуществляет поиск готового документа «Свидетельство о государственной регистрации программы для ЭВМ» и выгрузку выходных данных в таблицу.
- Подготовить электронный отчет (1 балл), в котором необходимо описать все используемые сторонние функции, также описать собственные переменные, функции, классы и т.д.
