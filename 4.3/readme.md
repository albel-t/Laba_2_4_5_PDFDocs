## PDF документы

- Необходимо написать программу, которая по имеющейся информации об авторах и названии осуществит поиск на [сайте ](https://www.fips.ru/iiss/search.xhtml)документа и при его наличии сохранит в таблицу по следующим столбцам: Название, Авторы, регистрационный номер, номер заявки, правообладатель, дата поступления, дата регистрации, ссылка на страницу документа
  - (1 балл) Осуществляется поиск по сайту по названию и/или по авторам
  - (1 балл) Осуществляется выгрузка страницы в `PDF` документ
